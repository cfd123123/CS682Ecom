Class: CategoryService

CategoryService

CategoryService manages all backend communication related to Categories.

new CategoryService()

Methods

deleteCategory(categoryID)

Sends a DELETE request to the backend for the Category by the given categoryID to be deleted from the database. A server response is returned back to the calling function. Only Employees and Admins have access to the backend mapping associated with this request.
Parameters:
Name Type Description
categoryID the ID of the category to be deleted
Returns:
the confirmation message from the backend
Type
Promise.<AxiosResponse.<T>>

getAll()

Sends a GET request to the backend for a list of all categories in the database. The server response containing the list of categories is returned back to the calling function.
Returns:
the list of all categories from the backend
Type
Promise.<AxiosResponse.<T>>

getCategoryProducts(categoryName)

Sends a GET request to the backend for all products associated with the given category. The server response containing the list of products is returned back to the calling function.
Parameters:
Name Type Description
categoryName the category name for which products are required.
Returns:
the list of products associated with the given category
Type
Promise.<AxiosResponse.<T>>

getListOfCategories(categoryIDs)

Sends a POST request to the backend with the given list of category IDs for a list of category objects corresponding to the given IDs, if they exist. The server response containing the list of category objects is returned back to the calling function.
Parameters:
Name Type Description
categoryIDs the list of category IDs
Returns:
the list of category objects
Type
Promise.<AxiosResponse.<T>>

saveCategory(category)

Sends a POST request to the backend to save the given category as a new category. The server response containing the saved category object is returned back to the calling function. Only Employees and Admins have access to the backend mapping associated with this request.
Parameters:
Name Type Description
category the category object to be saved
Returns:
the saved category object, which includes an ID
Type
Promise.<AxiosResponse.<T>>
