Class: Login

Login

Top level component that is the login page a user sees when they need to log in.

new Login()

Constructs this component with initially empty fields. Also binds the handleLogin, onChangeUsername, and onChangePassword functions to this component.

Members

username - the username to log in as
password - the password associated with username
loading - used to lazy load parts of this component
message - used to display a message to the user

handleLogin(event)

Initiates the login process after validating the user's input. If input is valid, this function calls the AuthService#login function, which saves the login token to localStorage, then routes the user to '/home'.
If input is not value, an error message is displayed to the user.
Parameters:
Name Type Description
event the event that triggered this function call
