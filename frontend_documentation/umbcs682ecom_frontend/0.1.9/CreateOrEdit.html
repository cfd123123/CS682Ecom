Class: CreateOrEdit

CreateOrEdit

This component is used to create or edit products. After the user is done entering product information, the details are sent to the backend for processing.
This component should only be accessible by employees and admins.

new CreateOrEdit()

Constructs this component with initial state values.

Members

addCategories

This function will trigger as the user types in the category field. If a comma is typed it will add the typed category to the array of categories

getExistingProduct

If a product is to be edited instead of created, this function calls the ProductService#getSingleProduct function to get the details of an existing product.

onChange

Processes when the user edits product information, updating this component's state.

onClose

If the user removes a category by clicking the X next to its name, this function removes that category from the array of categories.

onSubmit

Submits the entered information to the backed. If a new product is being created, the ProductService#addProduct function is called. If a product is being edited, the ProductService#updateProduct function is called instead.

parseCategories

Calls the CategoryService#getListOfCategories function to get the category names associated with the category ids stored in a product object.

state

name - the name of the product
shortDescription - a short description of the product
longDescription - a longer description of the product
price - the price of the product
quantity - how many of this product are available for purchase?
category - temporary variable to store a category name as it's entered by the user
categories - an array to store all categories to which this product belongs
categoryList - a temporary array to store user input of categories
image - a URL string pointing to an image of this product
