Class: CategoryHeader

CategoryHeader

Automatically lists all available product categories from the database. Individual categories can be clicked to trigger a categorical search.

new CategoryHeader()

Constructs this component with initial state variables.

Members

Methods

componentDidMount()

After the component mounts, this function calls the CategoryService#getAll function, which gets the list of categories from the backend, then updates this components state with the returned list.
