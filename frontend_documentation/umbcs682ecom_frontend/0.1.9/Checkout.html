Class: Checkout

Checkout

Top level component that is the checkout page a user sees after beginning the checkout process, but before placing their order. Users get to this page after clicking on the ProceedToCheckout button.

new Checkout(props)

Constructs this checkout component using the passed in props. The constructor also binds the placeOrder method to this component.
Parameters:
Name Type Description
props object the PreOrderResponse from the backend

Members

state

props.location.state.response:
> username - the user's username
> preOrder - the ID for the PreOrder containing the items in the user's cart
> cart - the set of product/quantity pairs in the user's cart
> products - the list of products in the PreOrder
> taxes - the taxes breakdown for the PreOrder
> shipping - the shipping charges breakdown for the PreOrder
> itemCount - the count of items in the PreOrder
> subTotal - the PreOrder's subtotal before taxes and shipping
> shippingCost - the PreOrder's total shipping cost
> taxCost - the PreOrder's total taxes
> total - the PreOrder's total price

Methods

placeOrder()

Calls the UserService#placeOrder function passing in the PreOrder ID. If placing the order is successful, this function moves the user to the OrderConfirmation page.
